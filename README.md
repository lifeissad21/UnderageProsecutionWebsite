# Underage Prosecution

Project Based Learning Project for AP Language and Composition.

This website presents an evidence-driven perspective on underage prosecution, including historical context, current practices, key issues, visual data, and references.

## Links

- Live website: https://underageprosecution.pages.dev
- GitHub Pages website: https://lifeissad21.github.io/UnderageProsecution/
- Repository: https://github.com/lifeissad21/UnderageProsecution

## Tech Stack

- Astro
- Tailwind CSS
- TypeScript

## Local Development

Run from the project root:

```sh
bun install
bun run dev
```

## Build and Preview

```sh
bun build
bun preview
```
